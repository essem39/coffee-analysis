 <div class="space-y-6">
 <div class="bg-white p-6 rounded-xl shadow-sm">
 <h3 class="text-xl font-bold mb-3">Чем отличается премиальный чай от обычного?</h3>
 <p class="text-gray-600">
 Премиальный чай производится из отборных листьев с высокогорных плантаций, собирается вручную 
 и проходит тщательную обработку. Он имеет более сложный вкусовой профиль и больше полезных веществ.
 </p>
 </div>

 <div class="bg-white p-6 rounded-xl shadow-sm">
 <h3 class="text-xl font-bold mb-3">Как правильно заваривать зелёный чай?</h3>
 <p class="text-gray-600">
 Используйте воду температурой 70-80°C (не кипяток!). Заваривайте 2-3 минуты. 
 Качественный зелёный чай можно заваривать 3-5 раз, каждый раз раскрывая новые оттенки вкуса.
 </p>
 </div>

 <div class="bg-white p-6 rounded-xl shadow-sm">
 <h3 class="text-xl font-bold mb-3">Почему премиальный чай дороже?</h3>
 <p class="text-gray-600">
 Высокая цена обусловлена ручным сбором, строгим отбором листьев, сложной обработкой 
 и ограниченными объёмами производства. Один килограмм лучшего чая может требовать 
 сбора десятков тысяч почек.
 </p>
 </div>

 <div class="bg-white p-6 rounded-xl shadow-sm">
 <h3 class="text-xl font-bold mb-3">Теряет ли чай свойства со временем?</h3>
 <p class="text-gray-600">
 Да, особенно зелёный и белый чай лучше употреблять в течение года после сбора. 
 Улуны и пуэры могут улучшаться с возрастом при правильном хранении. Чёрный чай 
 сохраняет качество 2-3 года.
 </p>
 </div>

 <div class="bg-white p-6 rounded-xl shadow-sm">
 <h3 class="text-xl font-bold mb-3">Можно ли пить чай на ночь?</h3>
 <p class="text-gray-600">
 Зависит от вида чая и вашей чувствительности к кофеину. Белый чай и травяные настои 
 подходят для вечера. Зелёный и чёрный чай лучше пить в первой половине дня. 
 Улун — универсальный вариант с умеренным содержанием кофеина.
 </p>
 </div>

 const menuBtn = document.getElementById('menu-btn');

 const menuOverlay = document.getElementById('menu-overlay');

 const mobileMenuLinks = document.querySelectorAll('.mobile-menu-link');

 mobileMenu.classList.add('open');
 menuOverlay.classList.add('open');

 }

 function closeMenu() {
 menuBtn.classList.remove('open');
 mobileMenu.classList.remove('open');
 menuOverlay.classList.remove('open');
 document.body.style.overflow = '';
 }

 menuBtn.addEventListener('click', openMenu);
 closeMenuBtn.addEventListener('click', closeMenu);
 menuOverlay.addEventListener('click', closeMenu);

 // Close menu when clicking on menu links
 mobileMenuLinks.forEach(link => {
 link.addEventListener('click', closeMenu);
 });

 // Smooth scrolling for anchor links
 document.querySelectorAll('a[href^="#"]').forEach(anchor => {
 anchor.addEventListener('click', function (e) {
 e.preventDefault();
 const target = document.querySelector(this.getAttribute('href'));
 if (target) {
 target.scrollIntoView({
 behavior: 'smooth',
 block: 'start'
 });
 }
 });
 });

 // Close mobile menu on window resize
 window.addEventListener('resize', () => {
 if (window.innerWidth >= 768) {
 closeMenu();
 }
 });
